HEX-редактор
Запустить hex.py. Первый параметр - директория редактируемого файла.
exit выход
next следующая страница
prev предыдущая страница
save сохранить
add pos v добавить на позицию знак или строку
rm pos [длина] удалить символы с определенной позиции
ch pos v изменить символ на позиции
Редактирование в hex
chH pos v
addH pos v
